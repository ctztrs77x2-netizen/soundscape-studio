# SoundScape Studio

An immersive ambient focus workspace for deep work, featuring:

- High-quality local ambient sounds (rain, thunder, ocean, etc.)
- 17 offline Lo-Fi tracks
- Procedural synthesizers (Brown Noise, Space Pads, Binaural Beats)
- Pomodoro timer with session tracking
- Notepad + Task checklist
- Beautiful glassmorphism UI + Zen Mode

## Installation (Development)

1. Clone this repo
2. Go to `chrome://extensions`
3. Enable "Developer mode"
4. Click "Load unpacked" and select this folder

## Privacy
